# WebCamPhotoBooth

WebCamPhotoBooth is an interactive web application that lets users apply live filters to their webcam feed and capture photos. Users can choose from multiple effects and download their snapshots directly.

## Features

- Access your webcam feed in real-time.
- Apply live filters including:
  - **Red Effect**
  - **RGB Split**
  - **Green Screen** (customizable with RGB sliders)
  - **Ghosting Effect**
- Capture photos and download them with a single click.
- Fully responsive layout optimized for desktop usage.

## How to Use

1. Open the project in a modern browser (Chrome, Firefox, Edge).
2. Allow access to your webcam when prompted.
3. Use the dropdown menu to select a filter.
4. Adjust RGB sliders for the Green Screen effect if desired.
5. Click **Take Photo** to capture a snapshot.
6. Your captured photos will appear below the webcam feed and can be downloaded.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- MediaDevices API
